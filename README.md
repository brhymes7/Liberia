# Change the Narrative - Nonprofit Website

A modern, responsive website for Change the Narrative, a nonprofit organization based in Liberia focused on community empowerment through education, healthcare, and economic development.

## Features

- **Modern Design**: Clean, professional layout using Tailwind CSS
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: TypeScript-powered interactions and animations
- **Accessible**: Built with accessibility best practices
- **Performance**: Optimized for fast loading and smooth user experience

## Color Scheme

- **Primary (Blue)**: #3b82f6 - Trust, stability, professionalism
- **Secondary (Red)**: #ef4444 - Urgency, passion, call-to-action
- **Tertiary (Green)**: #10b981 - Growth, hope, sustainability

## Sections

1. **Hero Section**: Compelling introduction with call-to-action
2. **About**: Mission, vision, and key statistics
3. **Programs**: Three main focus areas (Education, Healthcare, Economic Development)
4. **Impact**: Success stories and metrics
5. **Contact**: Contact information and inquiry form

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework
- **TypeScript**: Type-safe JavaScript for enhanced development
- **Font Awesome**: Icon library
- **Responsive Design**: Mobile-first approach

## Setup Instructions

1. **Clone or download** the project files
2. **Install dependencies** (optional, for TypeScript compilation):
   ```bash
   npm install
   ```
3. **Compile TypeScript** (optional):
   ```bash
   npm run build
   ```
4. **Serve the website**:
   ```bash
   npm run serve
   ```
   Or simply open `index.html` in your web browser

## Customization

### Colors
The color scheme can be easily modified in the Tailwind configuration within the HTML file. Update the `tailwind.config` object to change primary, secondary, and tertiary colors.

### Content
All content can be updated directly in the HTML file. Key areas to customize:
- Organization name and mission
- Program descriptions
- Contact information
- Statistics and impact numbers

### Styling
The website uses Tailwind CSS utility classes. Modify classes directly in the HTML or add custom CSS as needed.
